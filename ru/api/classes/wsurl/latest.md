# WSUrl

`WSUrl` разбирает переданный URL и предоставляет доступ к его основным частям: схеме, хосту, пути, query-строке, query-параметрам и сегментам пути.

## Поддерживает

- получение хоста;
- получение схемы URL;
- получение полного хоста со схемой;
- получение URL, сформированного классом;
- получение пути URL;
- получение URL без query-строки;
- получение исходной query-строки;
- получение query-параметра по ключу;
- получение части пути по индексу;
- получение типа записи из пути;
- получение последнего сегмента пути.

## Быстрый старт

```php
$url = new WSUrl('https://example.com/blog/post-name?foo=bar');

$host = $url->host();
$scheme = $url->scheme();
$path = $url->path();
$parameter = $url->getParameter('foo');
```

## Инициализация

### __construct($url)

Создаёт объект `WSUrl` на основе переданного URL.

Параметры:

| Параметр | Тип   | Описание                                                                         |
| -------- | ----- | -------------------------------------------------------------------------------- |
| `$url`   | mixed | URL, который передаётся в `urldecode()` и затем разбирается через `parse_url()`. |

Пример:

```php
$url = new WSUrl('https://example.com/catalog/item?sort=asc');
```

## Публичные методы

### host()

Возвращает хост URL.

Параметры: нет.

Возвращает:

| Тип   | Описание                                    |
| ----- | ------------------------------------------- |
| mixed | Значение хоста из результата `parse_url()`. |

Пример:

```php
$host = $url->host();
```

### scheme()

Возвращает схему URL.

Параметры: нет.

Возвращает:

| Тип   | Описание                                    |
| ----- | ------------------------------------------- |
| mixed | Значение схемы из результата `parse_url()`. |

Пример:

```php
$scheme = $url->scheme();
```

### fullHost()

Возвращает полный хост со схемой.

Параметры: нет.

Возвращает:

| Тип    | Описание                     |
| ------ | ---------------------------- |
| string | Строка вида `scheme://host`. |

Пример:

```php
$fullHost = $url->fullHost();
```

### url()

Возвращает URL, сформированный классом.

Параметры: нет.

Возвращает:

| Тип    | Описание                                                              |
| ------ | --------------------------------------------------------------------- |
| string | URL со схемой, хостом, путём и query-строкой, если она была передана. |

Пример:

```php
$currentUrl = $url->url();
```

### path()

Возвращает путь URL.

Параметры: нет.

Возвращает:

| Тип    | Описание                                                             |
| ------ | -------------------------------------------------------------------- |
| string | Путь без начального и конечного `/`, приведённый к нижнему регистру. |

Пример:

```php
$path = $url->path();
```

### fullPath()

Возвращает URL без query-строки.

Параметры: нет.

Возвращает:

| Тип    | Описание                         |
| ------ | -------------------------------- |
| string | Значение `url()` до символа `?`. |

Пример:

```php
$fullPath = $url->fullPath();
```

### query()

Возвращает исходную query-строку.

Параметры: нет.

Возвращает:

| Тип   | Описание                                  |
| ----- | ----------------------------------------- |
| mixed | Query-строка URL, если она была передана. |

Пример:

```php
$query = $url->query();
```

### getParameter($key)

Возвращает query-параметр по ключу после обработки через `Convert::noHTML()`.

Параметры:

| Параметр | Тип   | Описание              |
| -------- | ----- | --------------------- |
| `$key`   | mixed | Ключ query-параметра. |

Возвращает:

| Тип   | Описание                                  |
| ----- | ----------------------------------------- |
| mixed | Обработанное значение query-параметра.    |
| null  | Если параметр с таким ключом отсутствует. |

Пример:

```php
$value = $url->getParameter('page');
```

### getPathPart($key)

Возвращает часть пути по индексу.

Параметры:

| Параметр | Тип   | Описание           |
| -------- | ----- | ------------------ |
| `$key`   | mixed | Индекс части пути. |

Возвращает:

| Тип   | Описание                                      |
| ----- | --------------------------------------------- |
| mixed | Часть пути.                                   |
| null  | Если часть пути с таким индексом отсутствует. |

Пример:

```php
$section = $url->getPathPart(0);
```

### getOriginalParameter($key)

Возвращает исходное значение query-параметра без обработки через `Convert::noHTML()`.

Параметры:

| Параметр | Тип   | Описание              |
| -------- | ----- | --------------------- |
| `$key`   | mixed | Ключ query-параметра. |

Возвращает:

| Тип   | Описание                           |
| ----- | ---------------------------------- |
| mixed | Исходное значение query-параметра. |

Пример:

```php
$value = $url->getOriginalParameter('page');
```

### getPostType()

Возвращает тип записи, сформированный из частей пути, кроме последней.

Параметры: нет.

Возвращает:

| Тип   | Описание                                      |
| ----- | --------------------------------------------- |
| mixed | Строка с частями пути до последнего сегмента. |
| null  | Если тип записи не был сформирован.           |

Пример:

```php
$postType = $url->getPostType();
```

### getPageUrl()

Возвращает последнюю часть пути.

Параметры: нет.

Возвращает:

| Тип   | Описание                            |
| ----- | ----------------------------------- |
| mixed | Последний сегмент пути.             |
| null  | Если значение не было сформировано. |

Пример:

```php
$pageUrl = $url->getPageUrl();
```

### getBaseUrl()

Возвращает путь URL.

Параметры: нет.

Возвращает:

| Тип    | Описание                        |
| ------ | ------------------------------- |
| string | То же значение, что и `path()`. |

Пример:

```php
$baseUrl = $url->getBaseUrl();
```

## Форматы данных

### Query-параметры

Query-строка разбирается по символу `&`, затем каждый элемент разбирается по символу `=`.

| Элемент  | Описание                                      |
| -------- | --------------------------------------------- |
| Ключ     | Левая часть элемента query-строки до `=`.     |
| Значение | Правая часть элемента query-строки после `=`. |

### Части пути

Путь разбивается по символу `/`.

| Индекс | Описание                         |
| ------ | -------------------------------- |
| `0`    | Первая часть пути.               |
| `1`    | Вторая часть пути.               |
| `n`    | Следующая часть пути по индексу. |

## Исключения

В коде класса явные исключения не выбрасываются.

## Ограничения и замечания

* Статус класса не удалось подтвердить по коду или реестрам, поэтому указан `unknown`.
* Версия появления класса не указана, так как она не подтверждена кодом или реестрами.
* Конструктор ожидает, что `parse_url()` вернёт ключи `path`, `host` и `scheme`.
* Путь приводится к нижнему регистру через `mb_strtolower()`.
* Query-строка разбирается простым разделением по `&` и `=`.
* `getParameter()` возвращает значение после обработки через `Convert::noHTML()`.
* `getOriginalParameter()` обращается к query-параметру напрямую и не проверяет его наличие.
* Private-методы `initUrl()`, `generateQueryArray()` и `generatePathArray()` не входят в публичный API.
