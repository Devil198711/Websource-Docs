# Page

`Page` предоставляет статические методы для получения страниц CMS.

## Поддерживает

- получение страницы по идентификатору
- получение страницы по URL
- выбор только опубликованных страниц
- выбор страниц со свободной ссылкой
- преобразование метаданных страницы в массив значений

## Быстрый старт

```php
$page = Page::getPageByUrl('contacts', true);

if ($page) {
    echo Convert::sanitizeHtml($page['title']);
}
```

Создавать экземпляр класса не нужно.

## Получение страницы по ID

### getPage($id): array|false

Возвращает страницу по её идентификатору.

```php
$page = Page::getPage(15);

if ($page) {
    echo Convert::sanitizeHtml($page['title']);
}
```

Если страница не найдена, метод возвращает `false`.

Метаданные найденной страницы преобразуются в массив значений:

```php
$page = Page::getPage(15);

$subtitle = $page['meta']['subtitle'] ?? '';
```

Исходная структура метаданных сохраняется в `post_data`:

```php
$page = Page::getPage(15);

$metaDefinition = $page['post_data']['subtitle'] ?? [];
$metaValue = $page['meta']['subtitle'] ?? '';
```

## Получение страницы по URL

### getPageByUrl($url, $publish = false, $freeLink = false): array|false

Возвращает страницу по её URL.

```php
$page = Page::getPageByUrl('contacts', true);
```

Если страница не найдена, метод возвращает `false`.

### Параметры

| Параметр | Тип | Описание |
|---|---|---|
| `$url` | `string` | URL страницы без начального `/`. |
| `$publish` | `bool` | Если `true`, возвращается только опубликованная страница. |
| `$freeLink` | `bool` | Если `true`, страница должна поддерживать свободную ссылку. |

Для получения опубликованной страницы:

```php
$page = Page::getPageByUrl('contacts', true);
```

Для получения опубликованной страницы со свободной ссылкой:

```php
$page = Page::getPageByUrl('catalog', true, true);
```

В отличие от `getPage()`, метод `getPageByUrl()` возвращает поле `meta` в исходном виде, полученном из базы данных.

Для преобразования метаданных используйте `Convert::strToJson()`:

```php
$page = Page::getPageByUrl('contacts', true);

if ($page) {
    $meta = Convert::strToJson((string) ($page['meta'] ?? ''));
    $subtitle = $meta['subtitle']['value'] ?? '';
}
```

## Данные страницы

Методы возвращают массив с данными страницы.

| Ключ | Описание |
|---|---|
| `id` | Идентификатор страницы. |
| `title` | Заголовок страницы. |
| `content` | Основное содержимое. |
| `url` | URL страницы. |
| `img` | Путь к изображению страницы. |
| `author` | Идентификатор автора. |
| `date` | Дата создания. |
| `status` | Статус страницы. |
| `meta` | Метаданные страницы. |
| `template` | Название шаблона. |
| `seo_title` | SEO-заголовок. |
| `seo_description` | SEO-описание. |
| `seo_key` | Ключевые слова. |
| `parent` | Идентификатор родительской страницы. |
| `noindex` | Признак запрета индексации. |
| `canonical` | Канонический URL. |
| `free_link` | Признак поддержки свободной ссылки. |

Метод `getPage()` дополнительно добавляет ключ `post_data` с исходной структурой метаданных.

## Пример вывода страницы

```php
$page = Page::getPageByUrl('about', true);

if (!$page) {
    return;
}

$title = Convert::sanitizeHtml((string) ($page['title'] ?? ''));
$content = (string) ($page['content'] ?? '');

echo '<h1>' . $title . '</h1>';
echo $content;
```

Содержимое `content` может включать HTML, поэтому способ его вывода зависит от происхождения и назначения данных.

## Важные замечания

- перед использованием результата проверяйте, что страница найдена
- передавайте `true` в `$publish`, если нужна только опубликованная страница
- URL передаётся без начального `/`
- `getPage()` преобразует метаданные, а `getPageByUrl()` возвращает их в исходном виде
- экранируйте текстовые значения перед выводом в HTML
