# $WS_User

`$WS_User` содержит данные пользователя, авторизованного в текущем запросе.

Переменная доступна только при наличии действующей пользовательской сессии. Для неавторизованного посетителя она может быть не определена.

## Поддерживает

- получение идентификатора и логина пользователя
- получение адреса электронной почты и даты регистрации
- получение роли пользователя
- получение пути к аватару
- проверку наличия авторизованного пользователя

## Проверка авторизации

Перед обращением к данным проверяйте наличие идентификатора пользователя:

```php
if (isset($WS_User['id'])) {
    echo htmlspecialchars($WS_User['login'], ENT_QUOTES, 'UTF-8');
}
```

## Доступные данные

| Ключ | Описание |
|---|---|
| `id` | Идентификатор пользователя. |
| `login` | Логин пользователя. |
| `email` | Адрес электронной почты. |
| `register` | Дата регистрации. |
| `role` | Роль пользователя. |
| `avatar` | Путь к изображению аватара. |
| `activation` | Служебное значение состояния активации. |
| `restore` | Служебный код восстановления пароля. |

## Проверка роли

```php
if (($WS_User['role'] ?? '') === 'admin') {
    // Действие доступно администратору.
}
```

## Важные замечания

- перед использованием переменной проверяйте наличие нужного ключа
- экранируйте значения перед выводом в HTML
- не выводите `activation` и `restore` и не передавайте их клиенту
- метаданные пользователя хранятся отдельно в `$WS_UserMeta`
